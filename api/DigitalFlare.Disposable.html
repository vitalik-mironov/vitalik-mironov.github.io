    IsDisposeRequested

    Возвращает признак, указывающий был ли хотя бы один запрос выгрузки данного объекта (вызов метода Dispose()).

    Свойство не подвержено влиянию состояния выгрузки объекта.

    Следует учитывать, что после того, как поступил запрос выгрузки, модификация состояния объекта (такими методами как WriteDA<T>(ref T, T), UpdDABool<T>(ref T, Transform2<T>) и др.) приведет к вызову исключения ObjectDisposedException.

    UpdDA<T>(ref T, Transform<T>, out InterlockedUpdateResult<T>, Action<T>)

    Метод заменяет оригинальное значение в location новым значением, полученным от transform.

    В зависимости от наличия изменения в location после вызова функции transform эта функция будет вызвана повторно для актуального значения из location.

    Метод выполняет проверку выгрузки данного объекта при считывании из location, а также при записи в location, учитывая IsDisposeRequested.

    Declaration
    protected void UpdDA<T>(ref T location, Transform<T> transform, out InterlockedUpdateResult<T> result, Action<T> cleanup = null)
    where T : class
    Parameters
    Type Name Description
    T location Адрес переменной, в которой производится обновление.
    Transform<T> transform Функция трансформации оригинального значения в новое.

    Не может быть null.
    InterlockedUpdateResult<T> result Переменная, в которую будет помещен результат операции обновления.
    Action<T> cleanup Метод очистки результата функции transform.

    Вызывается только в тех случаях, когда результат функции не востребован:

    • начата выгрузка объекта и результат НЕ записан в location;

    • после вызова функции transform значение в location изменилось и функция transform должна быть вызвана повторно для актуального значения из location.

    UpdDABool<T>(ref T, Transform<T>, out T, Action<T>)

    Метод заменяет оригинальное значение в location новым значением, полученным от transform.

    Если оригинальное и новое значение идентичны, то метод возвратит false, иначе — true.

    В зависимости от наличия изменения значения в location после вызова функции transform эта функция будет вызвана повторно для нового значения из location.

    Метод выполняет проверку выгрузки данного объекта при считывании оригинального значения из location, а также при записи нового значения в location.

    Transform<T> transform Функция трансформации оригинального значения в новое.

    В зависимости от наличия изменения значения в location после вызова функции transform эта функция будет вызвана повторно для нового значения из location.

    Не может быть null.
    Action<T> transformResultCleanup Метод очистки результата вызова функции трансформации transform.

    Вызывается в тех случаях, когда результат функции трансформации не востребован:

    • начата выгрузка объекта;

    • после вызова функции transform значение в location изменилось и функция transform должна быть вызвана для изменённого значения в location.
    Returns
    Type Description
    Boolean Значение Boolean.

    Если значение в переменной location было заменено новым, полученным от transform, то возвращается true, иначе — false.
