    ConvertNullCond<TSource, TResult>(FluentApiUtilitiesHandle<TSource>, Func<TSource, TResult>)

    Конвертирует указанное значение (объект) hnd, используя функцию-конвертер converter.

    Если указанное значение есть null, то метод не вызывает функцию-конвертер и возвращает null.

    Если указанное значение может быть приведено к типу TResult, то метод не вызывает функцию-конвертер и возвращает приведенное к типу TResult указанное значение.

    Для других случаев метод возвращает результат функции-конвертера converter.
