    DisposableImplementation<TClass>

    Представляет общую реализацию выгружаемого объекта.

    Тип не предназначен для прямого использования в коде.

    Тип используется такими типами как Disposable, DisposeNotifying и др.

    Тип TClass:

    • Является классом;

    • Может иметь метод 'FireBeforeDispose(bool explicitDispose)'.

    • Может иметь метод 'FireAfterDisposed(bool explicitDispose)'.

    • Может иметь приватный метод 'P_Dispose(bool explicitDispose)' или неприватный 'Dispose(bool explicitDispose)', который реализует логику выгрузки.
