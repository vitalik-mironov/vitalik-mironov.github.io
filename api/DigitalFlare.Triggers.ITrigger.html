    Signal(ITriggerSignalProperties)

    Выполняет однократный перевод триггера в сигнальное состояние.

    Если на момент вызова метода триггер отключен (см. IsDisabled) или для триггера поступил запрос выгрузки (см. IsDisposeRequested), то перевод триггера в сигнальное состояние не выполняется.

    Не может быть null.
