    Class DisposableImplementation<TClass>

    Представляет общую реализацию выгружаемого объекта.

    Тип не предназначен для прямого использования в коде.

    Тип используется такими типами как Disposable, DisposeNotifying и др.

    Тип TClass:

    • Является классом;

    • Может иметь метод 'FireBeforeDispose(bool explicitDispose)'.

    • Может иметь метод 'FireAfterDisposed(bool explicitDispose)'.

    • Может иметь приватный метод 'P_Dispose(bool explicitDispose)' или неприватный 'Dispose(bool explicitDispose)', который реализует логику выгрузки.

    EnsureNotDisposeState(Boolean)

    Выполняет проверку на предмет состояния выгрузки данного объекта или наличия запроса на его выгрузку (см. IsDisposeRequested).

    Если для объекта была начата выгрузка или он уже выгружен (см. Disposing и IsDisposed), то вызывается исключение ObjectDisposedException.

    Если considerDisposeRequest == true и для объекта была запрошена выгрузка (см. IsDisposeRequested), то вызывается исключение ObjectDisposedException.

    UpdDA<T>(ref T, Transform<T>, out InterlockedUpdateResult<T>, Action<T>)

    Метод заменяет оригинальное значение в location новым значением, полученным от transform.

    В зависимости от наличия изменения в location после вызова функции transform эта функция будет вызвана повторно для актуального значения из location.

    Метод выполняет проверку выгрузки данного объекта при считывании из location, а также при записи в location, учитывая IsDisposeRequested.

    Declaration
    public void UpdDA<T>(ref T location, Transform<T> transform, out InterlockedUpdateResult<T> result, Action<T> cleanup = null)
    where T : class
    Parameters
    Type Name Description
    T location Адрес переменной, в которой производится обновление.
    Transform<T> transform Функция трансформации оригинального значения в новое.

    Не может быть null.
    InterlockedUpdateResult<T> result Переменная, в которую будет помещен результат операции обновления.
    Action<T> cleanup Метод очистки результата функции transform.

    Вызывается только в тех случаях, когда результат функции не востребован:

    • начата выгрузка объекта и результат НЕ записан в location;

    • после вызова функции transform значение в location изменилось и функция transform должна быть вызвана повторно для актуального значения из location.
