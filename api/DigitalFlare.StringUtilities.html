    IndentLines(String, String, String, Nullable<Int32>, Nullable<StringSplitOptions>, Boolean)

    Вставляет отступ в начале каждой новой строки текста inputText.

    Если значение inputText равно null или Empty, то метод возвратит значение inputText.

    Declaration
    public static string IndentLines(this string inputText, string lineSeparator = null, string indent = null, int? indentSize = default(int? ), StringSplitOptions? inputTextSplitOptions = default(StringSplitOptions? ), bool skipFirstLine = false)
    Parameters
    Type Name Description
    String inputText Входной текст.
    String lineSeparator Строка, определяющая разделитель строк в inputText.

    Если значение не задано или равно null, то будет использоваться значение NewLine.
    String indent Строка, определяющая отступ в начале каждой новой строки inputText.

    Если значение не задано или равно null, то будет использоваться символ табуляции (\u0009).
    Nullable<Int32> indentSize Размер отступа.

    Если не задан или равен null, то будет использоваться значение 1.
    Nullable<StringSplitOptions> inputTextSplitOptions Опции разбиения входного текста inputText на строки.

    Если не задан или равен null, то будет использоваться значение None.
    Boolean skipFirstLine Указывает, оставить ли первую строку текста inputText без отступа.
    Returns
    Type Description
    String Объект String.

    SubstringAfterLast(String, Char, ArgumentPlaceholder<String>)

    Возвращает подстроку после последнего вхождения указанного символа chr.

    Если входная строка есть null, то метод возвращает либо явно заданное значение defaultStr, либо str.

    Если последнее вхождение символа chr соответствует последнему символу входной строки, то метод возвращает Empty.

    Если символ chr отсутствует во входной строке, то метод возвращает либо явно заданное значение defaultStr, либо str.

    Declaration
    public static string SubstringAfterLast(this string str, char chr, ArgumentPlaceholder<string> defaultStr = default(ArgumentPlaceholder<string>))
    Parameters
    Type Name Description
    String str Входная строка.

    Может быть null.
    Char chr Символ.
    ArgumentPlaceholder<String> defaultStr Строка, возвращаемая в случае, если во входной строке указанный символ chr отсутствует, либо входная строка есть null.
    Returns
