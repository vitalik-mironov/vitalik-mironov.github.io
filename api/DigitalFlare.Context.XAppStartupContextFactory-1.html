    Delegate XAppStartupContextFactory<TDescription>

    Encapsulates a method that creates XApp startup context (see IXAppStartupContext).

    Can't return null.

    Namespace: DigitalFlare.Context
    Assembly: digital-flare.oxy.xapp.abstractions.dll
    Syntax
    public delegate Task<IXAppStartupContext<TDescription>> XAppStartupContextFactory<TDescription>(IXAppRuntimeService runtime, DescriptionLocator locator = null, TDescription description = null, ArgumentPlaceholder<XAppStartupContextHostHints> startupHints = default(ArgumentPlaceholder<XAppStartupContextHostHints>), ArgumentPlaceholder<IXAppContainerControl> containerControl = default(ArgumentPlaceholder<IXAppContainerControl>), IContext outerCtx = null)
    where TDescription : class, IXAppDescription;
    Parameters
    Type Name Description
    IXAppRuntimeService runtime XApp runtime service.

    Can't be null.
    DescriptionLocator locator App description locator (reference). See DescriptionLocator.

    Can be null.

    Must be null if description is not null.

    If locator and description are null, then locator or description, that be used by context to be created, should be determined by the method.
    TDescription description App description. See DescriptionLocator.

    Can be null.

    Must be null if locator is not null.

    If locator and description are null, then locator or description, that be used by context to be created, should be determined by the method.
    ArgumentPlaceholder<XAppStartupContextHostHints> startupHints Hints to the component, acting as XApp host, which behavior should be applied on app startup (see HostHints).
    ArgumentPlaceholder<IXAppContainerControl> containerControl XApp container control.
    IContext outerCtx Context, acting as an outer for the context to be created.
