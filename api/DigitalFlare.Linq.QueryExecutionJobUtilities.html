    ExecuteFullAsTaskContinuation<TResult>(IQueryExecutionJob<TResult>, Task, CancellationToken, TaskContinuationOptions)

    Выполняет задание исполнения запроса до тех пор, пока набор данных, соответствующий запросу, не будет получен полностью.

    Данный метод выполняет все запросы продолжения, которые могут быть получены в ходе выполнения указанного задания исполнения запроса.
