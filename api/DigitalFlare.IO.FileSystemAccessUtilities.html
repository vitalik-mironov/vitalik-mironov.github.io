    ZipArchiveFileAsync(FileInfo, FileInfo, Boolean, String, Boolean, IContext)

    Создает ZIP-архив файла sourceFile, сохраняет его по пути archiveFile.

    При deleteSourceFileOnSuccess = true, исходный файл удаляется.

    Declaration
    public static Task ZipArchiveFileAsync(FileInfo sourceFile, FileInfo archiveFile, bool deleteSourceFileOnSuccess, string overrideSourceFileName = null, bool overwrite = false, IContext ctx = null)
    Parameters
    Type Name Description
    FileInfo sourceFile Файл-источник архивирования.

    Не может быть null. Файл должен существовать.
    FileInfo archiveFile Файл-приёмник архивирования.

    Не может быть null.
    Boolean deleteSourceFileOnSuccess Флаг удаления файла-источника при успешном архивировании (исключения при удалении не выбрасываются).
    String overrideSourceFileName Переопределяет имя исходного файла в ZIP-архиве.

    Если не указано, то имя соответствует имени исходного файла sourceFile.
    Boolean overwrite Флаг перезаписи файла-приемника archiveFile, если такой существует.
    IContext ctx Контекст выполнения операции.
    Returns
    Type Description
    Task
