    ThrowIfCancellationRequested(IContext)

    Вызывает исключение, если для указанного контекста поступил запрос отмены (см. Ct(IContext, Boolean)).

    Наличие запроса отмены проверяется посредством токена отмены, ассоциированного с указанным контекстом (Ct(IContext, Boolean), IsCancellationRequested).

    Если ctx есть null, то проверка запроса отмены для контекста, очевидно, не производится.
