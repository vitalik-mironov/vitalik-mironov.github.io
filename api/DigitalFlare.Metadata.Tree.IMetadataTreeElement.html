    Interface IMetadataTreeElement

    Определяет элемент дерева метаданных.

    Элемент дерева метаданных — объект, представляющий связь метаданных (IMetadata, Metadata) и узла дерева (IMetadataTreeNode, Node), которому эти метаданные принадлежат.
