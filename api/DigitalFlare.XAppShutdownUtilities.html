    ShutdownAppAsync(IXApp<IXAppDescription>, Exception)

    Выполняет ShutdownAppAsync().

    Если ShutdownAppAsync() вызывает исключение, то данный метод оборачивает его и передает вызывающему коду. Кроме этого, если указано exception, то это исключение будет включено во внутренний стек передаваемого исключения.

    Declaration
    public static Task ShutdownAppAsync(this IXApp<IXAppDescription> app, Exception exception)
    Parameters
    Type Name Description
    IXApp<IXAppDescription> app Приложение.

    Может быть null. В этом случае возврат из метода производится немедленно.
    Exception exception Исключение, которое будет включено во внутренний стек исключения, вызванного данным методом.

    Может быть null.
