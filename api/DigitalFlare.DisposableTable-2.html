    Class DisposableTable<TKey, TValue>

    Disposable instance table.

    Main goal of this table likewise the idea of ConditionalWeakTable<TKey,TValue>, but instances leaves this table on special event — AfterDisposed.

    Built upon ImmutableDictionary<TKey,TValue>.
