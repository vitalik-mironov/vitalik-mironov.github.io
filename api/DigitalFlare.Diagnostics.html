    Для регистрации трансляторов см. RegisterTranslator(Type, IExceptionInfoTranslator) и UnregisterTranslator(Type, IExceptionInfoTranslator).

    IInnerExceptionInfoProvider

    Определяет свойство, представляющее информацию о внутреннем стеке исключений в виде ExceptionInfo.

    Используется при конструировании объекта ExceptionInfo для получения информации о внутреннем стеке исключения от объектов Exception, реализующих данный интерфейс.
