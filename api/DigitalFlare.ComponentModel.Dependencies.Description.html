    IDependenciesBindingsGroupDescription

    Представляет описание (конфигурацию) группы байндингов (связей) функциональных зависимостей.

    IDependenciesDescription

    Представляет описание (конфигурацию) (см. IDescription) функциональных зависимостей.

    IDependencyBindingDescription

    IDependencyBindingDescriptionBase

    Представляет базовое описание (конфигурацию) связи (байндинга) поставляемой функциональной зависимости (см. IDependencyBinding).

    IDependencyResolutionSettingsDescription

    Представляет описание (конфигурацию) дефолтных настроек разрешения функциональных зависимостей.

    IDependencySupportDescription

    Представляет описание (конфигурацию) компонента, поддерживающего определение функциональных зависимостей посредством описания IDependenciesDescription через свойство описания Dependencies.

    IXInstanceContractDependencyBindingDescription

    Представляет описание (конфигурацию) байндинга поставляемой функциональной зависимости (см. IDependencyBinding). При этом цель байндинга определяется XInstance-контрактами, определенными для заданного описания (см. DependencyTargetDescription).

    См. также XInstanceContractAttribute
