    OwnsValue

    Возвращает признак, указывающий "владеет" ли хранилище значением Value.

    Значение true указывает, что при выгрузке хранилища (см. Dispose()) значение Value также будет выгружено (если его тип реализует IDisposable).

    Обращение к свойству не подвержено влиянию состояния выгрузки.
