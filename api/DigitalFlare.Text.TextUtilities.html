    GetBraceTemplateNames(String)

    Возвращает массив непустых строк, представляющих "имена" подстановки, входящих во входную строку template.

    "Имена" подстановки во входной строке задаются как '{name}'.

    Пример. Строка 'Hello {world}! This {} {is} an {{example}}.' будет иметь следующий массив "имен" подстановки: "world", "is", "example".

    SubstituteBraceTemplate(String, Func<String, String>)

    Substitutes the values for the braced items.
    Declaration
    public static string SubstituteBraceTemplate(string template, Func<string, string> selector)
    Parameters
    Type Name Description
    String template The text containing substitution items in the form: '{substitionMarker1}, ... some text... {substition marker 2}, ... {substition marker N}'.

    Only items immediately braced are interpreted as the substitution markers.
    Func<String, String> selector The method returning the value of substitution.

    The first argument of the method is the substitution marker without braces.
    Returns
    Type Description
    String The result of substitution.

    Examples:

    'Hello, {substitution marker}' -> 'Hello, world'.

    'Hello, {{substitution marker}' -> 'Hello, {world'.

    'Hello, {}{substitution marker}' -> 'Hello, {}world'.

    'Hello, {}{substitution marker}}' -> 'Hello, {}world}'.

    'Hello, {}' -> 'Hello, {}'.

    public static Task<string> SubstituteTemplateAsync(string openToken, string closeToken, string template, ITextTemplateSubstitutionProvider values, InParamFunc<(string span, int spanStart, int spanLength, string variable), bool> filter = null, bool filterThrows = false, IContext ctx = null)
    Parameters
    Type Name Description
    String openToken The opening token of the variable's span.

    Can't be null, Empty or whitespace string.

    Length can't be greater than langword_csharp_8.
    String closeToken The closing token of the variable's span.

    Can't be null, Empty or whitespace string.

    Length can't be greater than langword_csharp_8.
    String template Template.
    ITextTemplateSubstitutionProvider values Maps the variable name to a value. That value will be placed instead of variable's span in the template.

    Can't be null.
    InParamFunc<ValueTuple<String, Int32, Int32, String>, Boolean> filter Special method to filter the variables to be substituted.

    true — substitute the variable, variable value will be taken from values method.

    false — skip the variable (see filterThrows).
    Boolean filterThrows Indicates whether an exception should be thrown when the variable don't match to the filter.

    false — the variable's span will be passed to result string as is.
