    AppendNewLineThen<T>(StringBuilder, T[], Func<T, String>)

    Добавляет в конец изменяемой строки сначала символы перевода строки (NewLine), а затем указанное содержимое в соответствии с указанным форматом.

    Данный метод операцию добавления выполняет с использованием распараллеливания. Т.е. исходный массив данных source разделяется на несколько секций и каждая секция обрабатывается в отдельном потоке выполнения.
