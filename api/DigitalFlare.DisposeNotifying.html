    FireBeforeDispose(Boolean)

    Метод, вызываемый перед явной или неявной выгрузкой.

    Не предназначен для прямого вызова из кода.

    Метод не вызывается после того, как один из вызывов был выполнен успешно.

    На момент вызова метода IsDisposeRequested == true.

    Перед вызовом базовой реализации FireBeforeDispose(Boolean) вызывает обработчик события BeforeDispose.
