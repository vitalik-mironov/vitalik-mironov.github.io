    GetOrAdd<TKey, TValue>(IDictionary<TKey, TValue>, TKey, Func<TKey, TValue>, Action<TValue>)

    Получает из словаря значение, соответствующее ключу key.

    Если в словаре соответсвующего ключу значения нет, то в словарь помещается значение, полученное от factory.

    Declaration
    public static TValue GetOrAdd<TKey, TValue>(this IDictionary<TKey, TValue> dictionary, TKey key, Func<TKey, TValue> factory, Action<TValue> unclaimedValue = null)
    Parameters
    Type Name Description
    IDictionary<TKey, TValue> dictionary Словарь.

    Не может быть null.
    TKey key Ключ.

    Не может быть null.
    Func<TKey, TValue> factory Метод-делегат, возвращающий значение, которое будет добавлено в словарь, если соответствующего указанному ключу значения нет.

    Вызывается единажды. Не вызвается вообще, если в словаре присутствует значение, соответствующее указанному ключу.

    Не может быть null.
    Action<TValue> unclaimedValue Метод-делегат, вызываемый, когда значение, полученное от factory, не было добавлено в словарь.

    Первый аргумент метода — значение, полученное от factory.
