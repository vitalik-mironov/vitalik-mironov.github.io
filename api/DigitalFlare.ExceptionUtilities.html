    IsOperationCancellationOf(Exception, CancellationToken)

    Определяет, является ли указанное исключение exception исключением отмены операции, с которой ассоциирован указанный токен отмены ct.

    Выполнение одного из условий для указанного исключения exception является необходимым:

    • Исключение имеет тип OperationCanceledException и его токен отмены равен указанному ct.

    • Исключение имеет тип AggregateException с единственным базовым исключением типа OperationCanceledException, токен отмены которого равен указанному ct.

    • Исключение имеет любой тип и его базовое исключение имеет тип OperationCanceledException а также токен отмены, равный указанному ct.
