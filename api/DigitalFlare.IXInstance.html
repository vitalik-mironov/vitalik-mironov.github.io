    RegisterInitialization(XInstanceInitialization)

    Регистрирует обработчик инициализации данного компонента.

    Регистрация обработчика возможна только до начала инициализации данного компонента, т.е. до перехода состояния State в InitializeSent. В случае, когда инициализация компонента уже была начата, выполнение метода вызовет исключение InvalidOperationException.
