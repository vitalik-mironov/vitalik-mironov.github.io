    Создает задачу ожидания завершения другой задачи.

    Следует учитывать, что созданная задача не транслирует конкретное финальное состояние ожидаемой задачи (отмена, результат или сбой).

    Declaration
    public static Task WaitCompletionAsync(this Task task, TaskCreationOptions options = TaskCreationOptions.None, CancellationToken ct = default(CancellationToken), bool cancellationBreak = false)
    Parameters
    Type Name Description
    Task task Ожидаемая задача.

    TaskCreationOptions options Опции создаваемой задачи ожидания.

    Boolean cancellationBreak Указывает поведение при поступлении сигнала отмены от ct:

    true — задача ожидания будет завершена;
